\doxysubsection{Detailed Description}
The Seven Tag Roster defined in the PGN standard is stored in the \mbox{\hyperlink{class_index_entry}{Index\+Entry}}, but 5 are indexes that refer to a \mbox{\hyperlink{class_name_base}{Name\+Base}} object. This helper struct stores the referred values.
