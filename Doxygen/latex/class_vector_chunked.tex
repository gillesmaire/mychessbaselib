\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T, size\+\_\+t CHUNKSHIFT$>$\newline
class Vector\+Chunked$<$ T, CHUNKSHIFT $>$}
A vector-\/like container. Not all the elements are stored contiguously like in a normal vector, but are allocated in separate chunks so that\+:
\begin{DoxyItemize}
\item growing in size do not move the elements and old references remain valid;
\item very large containers can be created. {\itshape CHUNKSHIFT\+:} is the base-\/2 logarithm of the number of T entries per chunk. Total size of a chunk\+: (2$^\wedge$\+CHUNKSHIFT)$\ast$sizeof(T) 
\end{DoxyItemize}
