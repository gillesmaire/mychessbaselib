{\footnotesize\ttfamily size\+\_\+t Filebuf\+::getline (\begin{DoxyParamCaption}\item[{char $\ast$}]{str,  }\item[{size\+\_\+t}]{count }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [inline]}}

Equivalent to std\+::fstream\+::getline, but faster (no sentry \mbox{[}27.\+7.\+2.\+1.\+3\mbox{]}).

Extracts characters from {\itshape $\ast$this} and stores them in successive locations of the array whose first element is pointed to by {\ttfamily str}, until the end of line (\textquotesingle{}\textbackslash{}n\textquotesingle{} char) or the end of file condition occurs. The \textquotesingle{}\textbackslash{}n\textquotesingle{} char is not stored into {\ttfamily str}, and a null character, even in case of errors, is appended.

Typical usage\+:
