\doxysubsection{Detailed Description}
class \mbox{\hyperlink{class_str_range}{Str\+Range}} -\/ parse a string interpreting its content as 1 or 2 integers separated by whitespace. The integers represent the min and max value of a range. If only one integer is provided it will represent both the min and max value.
