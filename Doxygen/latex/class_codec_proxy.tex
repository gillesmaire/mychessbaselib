\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Derived$>$\newline
class Codec\+Proxy$<$ Derived $>$}
Base class for non-\/native databases. Every class derived from \mbox{\hyperlink{class_i_codec_database}{ICodec\+Database}} must keep an {\itshape \mbox{\hyperlink{class_index}{Index}}} object and the corresponding {\itshape \mbox{\hyperlink{class_name_base}{Name\+Base}}} object fully updated in memory. This implies that the virtual function dyn\+\_\+open() must load in memory the header\textquotesingle{}s data of all the games; however a dependency between the codecs and the {\itshape \mbox{\hyperlink{class_index}{Index}}} class is not desirable. This class provides an interface that encapsulates the codecs, requiring only the ability to exchange {\itshape \mbox{\hyperlink{class_game}{Game}}} objects.
