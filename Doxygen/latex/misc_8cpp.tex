eco\+Reduce() -\/ maps eco to a smaller set 
\begin{DoxyParams}{Parameters}
{\em eco} & the eco value to convert (must be != 0)\\
\hline
\end{DoxyParams}
Scid ECO subcodes use 131 values for each canonical ECO. For example A00 is divided in A00,A00a,A00a1,A00a2,A00a3,A00a4,A00b...A00z4 corresponding to eco values 1,2,3,4,5,6,7...131 (value 0 means no ECO). This functions will map sub\+ECOs like A00a1...A00a4 into A00a, reducing the 131 values to 27.
