\doxysubsection{Detailed Description}
This class implements a PGN \char`\"{}visitor\char`\"{} that invokes the appropriate member functions of the associated \mbox{\hyperlink{class_game}{Game}} object for each type of PGN token. Errors are stored as \texorpdfstring{$<$}{<}line number, error message\texorpdfstring{$>$}{>} pairs; if the maximum number of errors is reached tokens are ignored until the end of the \mbox{\hyperlink{class_game}{Game}}.
