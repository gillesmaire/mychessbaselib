Detailed Description

template<class TElem, size_t UNDOMAX>
class UndoRedo< TElem, UNDOMAX >

A container useful for implementing a undo-redo behavior. UNDOMAX: max number of copies to store. Typical use: store(obj_ptr); // (1) modify obj; obj_ptr = undo(obj_ptr); // obj_ptr now contains the copy of (1)
