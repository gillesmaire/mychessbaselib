Detailed Description

template<class T, size_t CHUNKSHIFT>
class VectorChunked< T, CHUNKSHIFT >

A vector-like container. Not all the elements are stored contiguously like in a normal vector, but are allocated in separate chunks so that:

  • growing in size do not move the elements and old references remain valid;
  • very large containers can be created. CHUNKSHIFT: is the base-2 logarithm of the number of T entries per chunk. Total size of a chunk: (2^CHUNKSHIFT)*sizeof(T)
